# chrome-extension-translatorX
一个非常简洁的翻译插件

## 使用方法

通过命令`git clone https://github.com/jtaox/chrome-extension-translatorX.git`clone到本地，打开chrome->menu->更多工具->扩展程序页面，点击上方'加载已解压程序'，选择clone到本地的项目目录即可。

默认开启百度翻译
